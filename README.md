# Spear-to-The-Enemy
İstanbul Beykent Üniversitesi, Dijital Oyun Tasarımı bölümü, 1.sınıf öğrencilerinden Python yazılım dilinden yapılmış labirentten kaçış video oyunu.
A video game in which the player must escape a maze, created using the Python programming language by first-year students from the Digital Game Design department at Istanbul Beykent University.
